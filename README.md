### 📦 Setup Instructions

1. Ensure **PHP 8.2+** is installed. Check the version:
   ```bash
   php -v
   ```

2. Ensure **Composer** is installed. Verify it by running:
   ```bash
   composer -v
   ```
   If Composer is not installed, download and install it from **https://getcomposer.org/**.

3. Install project dependencies:
   ```bash
   composer update
   ```

4. Copy `.env.example` to `.env`, configure `APP_URL` and your database credentials, then run:
   ```bash
   php artisan key:generate
   php artisan migrate
   php artisan db:seed
   php artisan storage:link
   ```

5. Start the development server:
   ```bash
   php artisan serve
   ```

6. Open **http://127.0.0.1:8000** (or your configured domain) in your browser.
